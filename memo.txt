4. pc_cascadian #カスカディアン型惑星

6. pc_subarctic #亜北極海型惑星
# 窒素と酸素を主成分とする大気と寒く広大な水圏を持つ岩石の惑星。地表の90％以上は海洋と極冠に覆われ、険しいでこぼことしたまばらな島々が表面に点在している。そこには海洋生態系を支える豊富な火山活動と地熱活動が存在する。
7. pc_mangrove #マングローブ型惑星
8. pc_hajungle #高地ジャングル型惑星
9. pc_retinal #レチナール惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。海洋に隔てられた大陸を伴う活発で安定した水圏がある。植物はクロロフィルの代わりにレチナールデヒドを基にしており、そのため惑星は紫の色調という特色がある。
10. pc_forest #森林惑星
11. pc_wetswamp #森林湿地型惑星
12. pc_lake #湖沼惑星
13. pc_tepid #微温惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。海洋で隔てられた大きな陸地を持つ活発で安定した水圏がある。軸傾斜がほとんど無いこの惑星は、理想的な成長条件を備えており、様々な種類の光合成プロセスの進化を可能にしている。
14. pc_aquatic #水生惑星
# 窒素と酸素を主成分とする大気と広大な水圏を持つ岩石の惑星。惑星表面の全てが海洋に覆われている。
15. pc_tidallylocked #同期自転型惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。惑星が太陽に同期自転しており、惑星の片側は常に昼間であり、反対側は常に夜間にある。希薄な居住可能領域が夕暮れ地域に存在している。絶え間なく降り注ぐ日光と風が食料やエネルギー生成に理想的な環境を生み出している。
16. pc_floating #浮遊島惑星
# 窒素と酸素を主成分とする大気を持つ、奇妙な浮遊島の惑星。島は果て無き海洋世界の上空に吊り下げられているかのよう。密度の高い植生に多くの飛行生物がこの惑星には居住している。
17. pc_aquifer #帯水層型惑星
# 窒素-酸素の大気を持つ乾燥した岩石質の惑星。降水量は少ないが、この惑星には、地表の下や惑星上に散在する無数のラグーンに見られる豊かな水域がある。乾燥した熱い砂の上では生物はあまり育たないが、ラグーンや地下河川には生物が豊富に生息している。
18. pc_supercon #超大陸型惑星
# 窒素と酸素を主成分とする大気を持つ、草原と乾燥した砂漠に支配された岩石の惑星。この惑星の地殻変動は現在、超大陸の状態にある。沿岸部は緑豊かだが、内陸部の多くは草原であり、徐々に大きな乾燥した砂漠へと変化している。
19. pc_technoorganic ##テクノオーガニック惑星
# 窒素と酸素を主成分とする大気と安定した水圏を持つ岩石の惑星。高度に先進的な灌漑システムが惑星全体に張り巡らされている。
20. pc_biolumen #生物発光惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。大気中には風変わりだが無毒の粒子が存在し、惑星全域にわたって常に雲を形成している。こうした状況にもかかわらず、温室効果と地熱活動により、地表には生命が育まれている。ほぼ全ての動植物が生物発光の特性を備えている。
21. pc_ecocity #エコアーコロジー
# 世界は都市化と産業化の無秩序な拡大に呑み込まれ、一部は完璧な手入れをされた庭園に覆われている。自然の岩石層はもはや軌道上から認識することすらできない。
22. pc_lichen #苔惑星
# 窒素-酸素の大気を持つ冷たくて岩の多い世界。この世界は極めて安定した水域を持ち、一般的に予測可能な低温であるため、ごつごつした山や氷河の間に惑星全体に広がる巨大な地衣類の森が生育している。
23. pc_primal #原始惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。この若い惑星の火山活動は減速し、複雑な動植物を生み出している。それが通気性のある大気を作り出して、水圏を安定させるのに役立っている。
24. pc_megafloratol #メガフローラ惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。森林が密集しており、水圏が強固で健全である。この惑星には地表を覆う巨大なメガフローラが繁茂している。根は軌道上からも観測できるほど大きい。巨大な植物に不慣れな種族には、この地の居住性は最悪と言える。
25. pc_milarc #要塞アーコロジー
# 軍需産業の無秩序な広がりに完全に飲み込まれた惑星。自然の岩盤は軌道上からはもはや見分けがつかず、高密度の超高層都市や浮いている乾ドック、過酷な試験場や訓練場で完全に覆われている。

7. pc_mesa #卓上台地型惑星

8. pc_hadesert #高地砂漠型惑星
# 窒素と酸素を主成分とする大気を持つ、冷たく乾燥した岩石の惑星。極度の高度の岳が惑星のほとんどは雨陰を引き起こし、塩類湖底と乾燥した渓谷が平地の大半を占める。水圏は僅かな湖と山の泉にある。
9. pc_prairie #プレーリー型惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。巨大な内海を包み込む単一の大陸を持つ季節性の水圏。乾燥した夏季と厳しい寒さの冬季が対照的である。
10. pc_med #地中海型惑星
11. pc_outback #高湿乾燥型惑星
# 窒素と酸素を主成分とする大気を持つ、乾燥した暑い惑星。地表の大半は乾燥、または半乾燥した低木地帯か砂漠だが水辺の周囲には他の植生もみられる。水圏の大部分は内陸の海か自然の地下泉に限られている。
12. pc_veld #ベルト型惑星
# 窒素と酸素を主成分とする大気を持つ、乾燥した岩石の惑星。この亜熱帯の風景は、低木の林と開けたサバンナに覆われている。この惑星は穏やかな冬と長く暑い夏の気候を持つ。
13. pc_semiarid #半乾燥型惑星
# 窒素と酸素を主成分とする大気に覆われたスカブランドと岩だらけの谷間に支配された岩石の惑星。水圏のおかげで短いながらも重要な雨季があるが、どこにでもある草を除けば、植生は保護された谷間や大きな水域の近くに集中している。
14. pc_coral #珊瑚惑星
# 窒素と酸素を主成分とする大気を持つ、乾燥した岩石の惑星。かつての海洋は磁気圏の弱さと他の生態学的要因で失われており、残された水域はミネラル塩分が非常に濃く僅かに毒性がある。この惑星の海に生息していた珊瑚は乾燥した環境に適応し、この惑星の至る所で森林として見つけることができる。
15. pc_salt #塩結晶型惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。鉱物に覆われた地形は、大部分が塩の平原と山脈、そして大規模な結晶層で構成されている。地表の大半は動植物が住むのに適さない。
16. pc_karst #カルスト惑星
# 窒素と酸素を主成分とする大気を持つ岩石の惑星。磁場の弱いこの世界では地表の大半は不毛なカルスト地形である。一方で地表下には途切れることのない洞窟群が豊富な水圏とともに広がっている。

14. pc_duneforest #砂丘森型惑星
# 窒素と酸素を主成分とする大気を持つ、冷たい山岳の惑星。霧が多く、降水量が少ないため、低地の多くは乾燥した砂丘となっており、雪がほとんど降らないため森林が山頂を覆っている。
15. pc_lanthanide ランタノイド型惑星
# 窒素と酸素を主成分とする大気に覆われた山岳惑星。希土類金属と磁気活動が異常に高いため、リング状の山頂には雪が積もっている。季節の変化は少ないが、まだ液体の水があるため丈夫な植物を育てることができる。
16. pc_rogue #変異惑星
# 窒素と酸素を主成分とする大気を持つ、凍てついた岩石の惑星。この惑星は星系外で捕獲されたもので、長い年月をかけて宇宙空間を移動した末にこの星系に辿り着いた。極度の地熱活動により、地表のエネルギーが届く場所に生命が存在している。この活動はまた惑星の発する熱を閉じ込めるのに十分な厚さの大気を引き起こしている。
17. pc_storm #暴風惑星
# 窒素と酸素を主成分とする大気を持つ、冷たく湿った岩石の惑星。惑星の主要な陸地はどちらかの極にあり、年間を通じて強烈な嵐を発生させている巨大な単一の海洋に分断されている。技術的には居住可能であるが、これらの土地で支えられるのは、少数の丈夫な動植物だけであり、惑星上の生命の大多数は、水面下に存在している。
18. pc_glacio #氷河火山型惑星
# 窒素と酸素を主成分とする大気を持つ、山岳地帯の凍てついた惑星。火山活動が活発でなければ、氷河と氷の惑星になっていただろうが、代わりに多くの火山が熱い湖を作り、そこにたくましい生命がしがみついている。
19. pc_geothermal #地熱北極型惑星
# 氷の広がりが熱帯の丘を別け、自然の地熱通気口によって熱せられた窒素と酸素を主成分とする大気を持つ、寒冷な惑星。水圏は溶融水の中にあり、通気口から蒸気が吹き上げている。
20. pc_eyeball #眼球惑星
# 窒素と酸素を主成分とする大気を持つ、凍った湿潤な岩石の惑星。親星に固定された明るい側は、常に太陽の光が降り注ぐ解凍された海であり、暗い側は凍った氷河で構成されている。この二つの異なる境界では、強風と強力な海流が発生している。
